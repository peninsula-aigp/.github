## 🏁 What Is This?

We're a group of students (ages 12–18) from [Peninsula Baptist Church](https://peninsulabaptist.com) in Mooresville, NC competing in the **[AI Grand Prix](https://theaigrandprix.com)** — a global autonomous drone racing competition by Anduril Industries with a $500,000 prize pool.

Our teams write Python code that flies drones through racecourses with **zero human control**. No pilots. Pure software.
